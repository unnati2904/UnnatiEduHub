## 🚀 About The Project

UnnatiEduHub is a responsive educational platform designed to provide learners with an engaging and user-friendly experience.

The website focuses on clean design, accessibility, responsiveness, and modern web development practices. It adapts seamlessly across desktops, tablets, and mobile devices.
