## 📖 About The Project

**UnnatiEduHub** is a responsive educational platform built using **HTML5** and **CSS3**. The project focuses on creating a clean, modern, and accessible learning environment with intuitive navigation and responsive layouts.

The website is optimized for desktops, tablets, and mobile devices, ensuring a seamless user experience across different screen sizes.
